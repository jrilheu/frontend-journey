Clase 1 – JavaScript esencial: variables, tipos y operadores (introducción)

Duración: 35–45 minutos

Objetivo de esta clase

Este módulo tiene 3 clases sobre variables, tipos de datos y operadores en JavaScript. Si es tu primer contacto con el lenguaje, aquí no asumimos nada: verás qué es JS, cómo probar código en la consola, cómo guardar valores en variables y cómo combinarlos con operadores. Las clases 2 y 3 profundizarán en cadenas, números, conversiones y más matices.

¿Qué es JavaScript y para qué sirve?

JavaScript es un lenguaje de programación que, en el navegador, permite que las páginas hagan cosas: reaccionar a clics, validar formularios, cargar datos, animar interfaces… También se usa fuera del navegador (por ejemplo con Node.js) para herramientas y servidores.

En esta ruta te centras en el JS que acompaña al HTML y CSS que ya conoces: es la tercera pieza del frontend clásico.

Cómo probar lo que escribas

La forma más rápida al empezar es abrir las herramientas de desarrollo del navegador (F12 o clic derecho → Inspeccionar), ir a la pestaña Consola y escribir líneas sueltas. También puedes crear un archivo .html con un <script> o un .js enlazado; lo importante es tener un lugar donde el motor de JS ejecute tu código.

console.log("Hola, mundo");

console.log muestra un valor en la consola: es tu primera herramienta para ver qué está pasando dentro del programa.

Variables: guardar y reutilizar valores

Una variable es un nombre que apunta a un valor en memoria. Así no repites números ni textos largos y puedes actualizar el valor cuando cambie la situación.

let y const

En JavaScript moderno se usan sobre todo dos palabras clave:

  • let — el valor puede cambiar más adelante (reasignación).
  • const — no puedes reasignar la variable a otro valor (sigue siendo útil para “constantes” lógicas y para objetos que no reemplazas por otro).
let contador = 0;
contador = contador + 1;

const tituloDelCurso = "JavaScript esencial";
// tituloDelCurso = "Otro"; // esto daría error

¿Y var?

var es la forma antigua; tiene reglas de alcance que suelen confundir al principio. En código nuevo se prefiere let y const. Si ves var en tutoriales viejos, puedes mentalmente traducirlo a let salvo casos raros.

Nombres legibles

Usa camelCase para variables (precioTotal, nombreUsuario). Evita abreviaturas crípticas; el nombre debe ayudar a leer el código en voz baja.

Regla mental: por defecto usa const. Si necesitas reasignar, cambia a let. Así reduces cambios accidentales.

Tipos de datos (visión general)

JavaScript es de tipado dinámico: no declaras el tipo en la variable; el motor lo infiere según el valor que guardas. Los tipos primitivos más habituales al inicio son:

  • string — texto entre comillas ("hola" o 'hola').
  • number — números enteros y decimales (42, 3.14).
  • boolean — true o false.
  • undefined — valor que indica “aún no asignado” (por ejemplo una variable declarada sin valor).
  • null — ausencia intencionada de valor (“aquí no hay objeto” por decisión).
  • bigint — enteros muy grandes (menos frecuente al principio).
  • symbol — identificadores únicos (tema avanzado; lo dejamos para más adelante).

Con typeof puedes preguntar qué tipo “cree” el motor que es una expresión:

typeof "hola";   // "string"
typeof 42;         // "number"
typeof true;       // "boolean"
typeof undefined;  // "undefined"

Hay curiosidades (por ejemplo typeof null devuelve "object" por razones históricas); en la Clase 2 veremos null y conversiones con más calma.

Operadores esenciales

Aritméticos

Suma +, resta -, multiplicación *, división /, resto % (módulo). El operador + también concatena strings:

"Hola, " + "Ana";  // "Hola, Ana"
10 + " años";       // "10 años" (el número se convierte a texto)

Asignación

= asigna. Existen atajos como +=, -=, *=, /=:

let x = 5;
x += 2;  // x vale 7

Comparación

Para comparar valores, en código moderno casi siempre querrás la igualdad estricta === y la desigualdad estricta !==: comparan valor y tipo sin “trucos” de conversión.

5 === "5";   // false
5 == "5";    // true (evita == al principio; puede sorprender)

Lógicos

&& (y), || (o), ! (no) combinan condiciones booleanas y aparecen mucho en if y en expresiones condicionales (más adelante en el curso).

true && false;  // false
true || false;   // true
!true;           // false
Consejo: cada vez que aprendas algo nuevo en JS, pruébalo en la consola con console.log. Ver el resultado real quita dudas más rápido que solo leer la teoría.

Comprueba lo aprendido

Práctica (15–20 min):

  1. Declara con const tu nombre y tu ciudad; muestra ambos con console.log.
  2. Declara con let un contador en 0; súmale 1 dos veces y muestra el resultado.
  3. Escribe tres expresiones y usa typeof sobre cada una (por ejemplo un string, un número y un booleano).
  4. Prueba === entre un número y el mismo número como string; repite con == y anota la diferencia.

En la Clase 2 profundizaremos en strings, números, null, undefined y conversiones entre tipos.
